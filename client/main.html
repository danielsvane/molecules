Hydrogen wave function visualization

Generates hydrogenic wave functions from the n, l and m quantum numbers. The wave is visualised using the square modulus (probability) function, by taking random points inside the sphere marked by black dots, and calculating the probability. The probability is represented as a white dot with varying transparency. You can zoom and rotate using the mouse.

The source code is available here.

Options

Select n, l and m quantum numbers for the wave function. Change bounding radius (black dots) to limit where the particles are generated. Increase the amount of particles for better/worse visualization, maximum is around 10000 particles.
